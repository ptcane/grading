Practical Data Analysis With Python

Learn to use Python to collate, correlate, and communicate your data.
learn@repl.it

Students will acquire knowledge and skills in the use of Python for tackling practical data analysis projects from start to finish. They will be able to gather data from a variety of sources, process and analyze it to gain new insights, and communicate those findings to others.

Who is this course for?

Students may have previous knowledge or experience in using spreadsheet applications such as Excel or Google Sheets for data analysis but want to be able to automate tasks and work with larger datasets including using databases and web APIs, while developing their Python programming skills.

Requirements

Before taking this course, students should have a basic knowledge of Python programming, including an understanding of the main data structures (lists, dictionaries), conditions (if, else), iteration, and functions; please review the Intro to Python curriculum and content if in doubt.

Collecting, cleaning, and combining data

  • Working with CSV, Excel, and JSON files; interacting with databases (? replit, SQL); getting data from web APIs; joining and merging datasets
  • Finding errors and missing values; removing and updating values; extracting new features from raw data; conversion between different data and file types

Aggregating and analyzing data

  • Grouping datasets by values in given field; comparing aggregate statistics of different groups; summarizing the distribution of values
  • Understanding relationships through calculation of correlation coefficients, linear regression and confidence intervals

Sharing and visualizing data

  • Creating charts and maps; automation of file processing and reports; developing interactive charts and web-based dashboards

The course comprises of 6 sessions, each of which have the following components:

Slides

Slide decks are provided in the following formats:

  • A reveal.js slideshow, which can be used in any web browser
  • A Jupyter notebook, designed for use with the RISE package

Using RISE allows the teacher to modify and execute code cells within the presentation on-the-fly, as well as having supplementary notes for each slide. The slides are authored in Jupyter notebook and so the code and notes can be shared with students.

Exercises

Students will be given coding tasks, such as defining functions and creating objects with particular attributes, which require use the techniques covered in the slides. Their solutions will be submitted to and assessed using an auto-grading system on repl.it.

Students will be provided with feedback on their errors so that they can attempt to make corrections and re-submit. Teachers will be able to see which tasks students had difficulty with.

Resources

Students will be provided with links to relevant documentation, web pages and videos through which they can broaden their understanding of the topics covered in the session.

Students will be able to complete the tasks either within a notebook (.ipynb file) or the associated script (.py) file. Upon submission to repl.it, the associated files will be automatically synchronized, so that the student can subsequently work on the other file if they prefer.

Each session will be based around a practical, end-to-end analysis of a different dataset. Each session will build upon the sessions already completed, using the skills previously learned.

The overview below indicates the new techniques to be introduced during each session.

Session 01

  • Overview of Pandas and DataFrames
  • Reading and writing CSV files
  • Understanding a dataset
  • Filtering DataFrames

Session 02

  • Dealing with missing values and errors
  • Pandas data types and type conversion
  • Creating new columns and feature extraction
  • Grouping data and comparing groups

Session 03

  • Plotting with Pandas and Seaborn
  • Creating interactive charts with Altair

Session 04

  • Working with JSON files
  • Fetching data from Web APIs

Session 05

  • Correlation, regression, and confidence intervals

Session 06

  • Creating an interactive web-based dashboard
